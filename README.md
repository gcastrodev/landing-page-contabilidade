# 📊 ContaPro Contabilidade

Landing Page moderna, responsiva e otimizada para escritórios de contabilidade, desenvolvida com **HTML5, CSS3 (Design System com Variáveis) e JavaScript Vanilla**.

Projeto ideal para:
- Escritórios contábeis
- Empresas de consultoria
- Profissionais liberais
- Templates comerciais
- Portfólio profissional

---

## 🚀 Demonstração

> Projeto estático pronto para hospedagem em:
- GitHub Pages
- Vercel
- Netlify
- Hostinger
- Qualquer servidor estático

---

## 🧱 Tecnologias Utilizadas

- HTML5 Semântico
- CSS3 com Variáveis (Design System)
- Flexbox & CSS Grid
- JavaScript Vanilla (ES6+)
- IntersectionObserver API
- Scroll Animations
- Responsividade Mobile-First
- Boas práticas de performance

---

## 🎨 Design System

O projeto utiliza variáveis CSS organizadas em:

- 🎨 Cores primárias e secundárias
- 🌓 Cores neutras
- 🔤 Tipografia
- 📏 Espaçamentos
- 🌫 Sombras
- 🔵 Border Radius
- ⚡ Transições
- 📦 Largura máxima do container

Tudo centralizado no `:root`, facilitando customização.

---

## 📱 Responsividade

Breakpoints implementados:

- 🖥 Desktop (1200px+)
- 💻 Tablet (até 1024px)
- 📱 Mobile (até 768px)
- 📱 Small Mobile (até 480px)

Layout adaptado com:
- Grid responsivo
- Menu mobile lateral
- Seções reorganizadas
- Botões full-width no mobile
